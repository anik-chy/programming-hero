# Module 3
## To check the installation

``` bash
    git --version
```

## To create a new local repo (Quick setup)
``` bash
    git init
    git add .
    git commit -m "first commit"
    git branch -M main
    git remote add origin 'repo-link'
    git push
```

## To push files
``` bash
    git add .
    git commit -m "Message"
    git push
```
